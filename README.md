# Wanderer Travel Blog

A clean and responsive travel blog article page built using HTML, CSS, and JavaScript.

## Features

- Beautiful hero section with background image
- Article layout with left content and right sidebar
- Author profile card
- Interactive tags (clickable)
- Responsive design (mobile-friendly)

## Technologies Used

- HTML5
- CSS3 (Flexbox)
- JavaScript (DOM manipulation)

## Preview

A modern travel blog UI featuring an immersive hero image and structured article layout.

## How to Use

1. Clone the repository
2. Open `index.html` in your browser
